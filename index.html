IBM Design Principal Application

Created a digital application for IBM client working to promote themselves within IBM heirarchy. Done when the standard was still paper materials as backup for a PDF application.
more

Client approached me over application format: standard expectations at the time was a 'smart' pdf with backup paper materials provided. This traditional application process required the HR and MGMT group to keep track of multiple formats per each applicant. Materials were cross checked in case of PDF issues and each applicant took a significant time investment to review. We developed a digital application in web form that utilized IBM visual language; hosted on their secure web infrastructure (bluemix/git). The application was praised for its tech integration as well as the overwhelming simplicity compared to most pdf/paper applications. There was no need to carry around physical materials or keep a local copy of the application downloaded- the reviewers were given a log in and were then able to access the application from anywhere on any machine. The application process was such a stark contrast to the norm that future applications were requested in web accessible format.
