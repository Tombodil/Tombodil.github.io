Truckersmall.com

Online Trucking source and drop sale site. Original site development and initial wordpress environment configured by Vouelger Group Inc. Additional layout, menu, text and UI work completed by me over three months. Created multiple sub pages with their page builder and utilized php and html code in order to add and modify functionality of the carts, shipping calculations and other details that were best kept as second layer (DOM) dynamics rather then CMS handled statics. Blog posts were mine as well. Pages and blog posts were optimized for SEO which was tracked with online tools and driven by regional serch statistics and industry leading goods, services and purchases. Worked with remote marketing team to facilitate digital ecology including facebook, twitter and instagram feed with syndicated content to company specifications.

IBM Design Principal Application

Created a digital application for IBM client working to promote themselves within IBM heirarchy. Done when the standard was still paper copies as backup for a PDF.

TropeCollaborative.com

Third generation personal representation of client group. Site base changed to WIX as expeditionary test of database and data handling integrated CMS. Some tricks and layout hacks inplemented to counter some of the more limited layout and formatting options available at start. Results obtained in spite of lack of CSS editing.

25Years

Anniversary site with a memorial note for one for one of the client's parents. Jquery enabled HTML with multi-selector CSS states that simplify the Jquery/Javascript load. All interaction state visual changes done with CSS classes while animation and data states accomplished with JQuery.

Kallish Holiday Site (yearly)

Yearly holiday site for Clients' family. Images and layout iterated each year. Grid layout with adjustment and detail views. Detail view tagged with format so that different aspect ratios are aligned to different grid layouts. Features built on top of core Masonry plugin layout.
